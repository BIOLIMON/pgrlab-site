# pgrlab-site

Sitio web oficial del **Laboratorio PGR** (Plant Growth Regulators), construido con TypeScript. Presenta las líneas de investigación del laboratorio, publicaciones, equipo y recursos relacionados con la ciencia de plantas y genómica vegetal.

## Instalación

```bash
npm install
npm run dev
```

## Scripts disponibles

| Comando | Descripción |
|---|---|
| `npm run dev` | Servidor de desarrollo |
| `npm run build` | Build de producción |
| `npm run preview` | Vista previa del build |
